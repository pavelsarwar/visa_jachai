সৌদি আরবএর ভিসা

ব্রাউজারের এ্যাড্রেস লেখার ঘরে https://www.eserve.com.sa/VVSWeb/ লিখে কী-বোর্ডে এন্টার বাটনে চাপ দিয়ে (চিত্র ১)ওয়েবসাইটে প্রবেশ করুন।


এখানে English লেখার উপর ক্লিক করলে (চিত্র ২) ওয়েবসাইটটি ইংরেজীতে দেখা যাবে (চিত্র ৩)।

Please Choose The First Value ক্লিক করে ড্রপ ডাউন মেনু থেকে Visa Number বা Passport Number সিলেক্ট করে পাশের ফাঁকা ঘরে Visa Number বা Passport Number লিখুন (চিত্র ৪)।

একইভাবে Please Choose The Second Value ক্লিক করে ড্রপ ডাউন মেনু থেকে Name বা Date of Birth সিলেক্ট করে পাশের ফাঁকা ঘরে Name বা Date of Birth লিখুন (চিত্র ৫)।


Nationality এর ঘরের পাশের কালো ছোট ত্রিভুজে ক্লিক করে নিজের দেশের নাম (যেমন Bangladesh) সিলেক্ট করে Check বাটন ক্লিক করুন (চিত্র ৬)। ভিসাটি সঠিক নাহলে Error মেসেজ আসবে (চিত্র ৭)।
