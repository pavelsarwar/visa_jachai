ওমানের ভিসা

ব্রাউজারের এ্যাড্রেস লেখার ঘরে http://www.rop.gov.om/VisaAppStatus/english/ লিখে কী-বোর্ডে এন্টার বাটনে চাপ দিয়ে ওয়েবসাইটে প্রবেশ করুন। নীচের উইন্ডোটি আসবে (চিত্র ১)।

ভিসার আবেদন ম্যানুয়ালি করে থাকলে Application Number আর অনলাইনে আবেদন করে থাকলে Web Application Number লিখুন। ৩ নম্বর ঘরে পাসপোর্ট নম্বর লিখতে হবে।

৪ নম্বর ঘরে ডান পাশের ছোট কালো ত্রিভুজে ক্লিক করলে একটি ড্রপ ডাউন মেনু আসবে। সেখান থেকে Passport Country হিসাবে দেশের নাম সিলেক্ট করুন। এরপর Submit বাটন ক্লিক করলে ভিসার অবস্থা জানা যাবে।
